# 全世界只剩我会古法编程

这是一个在线小说阅读器项目，使用 GitHub Pages 托管。

## 📖 在线阅读

**https://kings0527.github.io/the-last-hand-coder/**

## 🚀 功能特性

- 📚 目录导航 - 快速跳转到任意章节
- ⬅️➡️ 上一章/下一章 - 便捷的阅读导航
- ⌨️ 键盘快捷键 - 方向键翻页
- 💾 阅读进度保存 - 自动记录上次阅读位置
- 📱 响应式设计 - 支持移动端阅读

## 📂 目录结构

```
.
├── index.html          # 主页面（阅读器）
├── index.md            # 封面介绍
├── cover.jpg           # 封面图片
├── novel/
│   └── chapters/       # 80章小说内容
│       ├── chapter-01.md
│       ├── chapter-02.md
│       └── ...
└── README.md           # 本文件
```

## 🔧 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS (CDN)
- Marked.js (Markdown 渲染)
- GitHub Pages (静态托管)
